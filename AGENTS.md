# AGENTS.md

## Project

Single-package Python library: `histoprint/` — pretty-prints NumPy (and other) histograms to the terminal using Unicode combining characters and color codes. Entry point CLI: `histoprint/cli.py`.

## Commands

- **Lint + format**: `pre-commit run --all-files` (runs ruff, ruff-format, mypy, plus standard hooks)
- **Tests**: `pytest -s` (from repo root; installs via `pip install -e .[test,boost,uproot,rich] pandas`)
- **Mypy only**: `pre-commit run mypy --all-files`
- **Nox**: `nox` runs `lint` and `tests` sessions (nox uses uv backend)
- **Build**: `nox -s build` or `python -m build`

## Order

Lint → typecheck → test. CI runs pre-commit separately from pytest.

## Key details

- Python ≥3.10, tested 3.10–3.14. CI uses `uv pip install --system`.
- `version.py` is auto-generated by hatch-vcs (git tags). Don't hand-edit.
- Tests live in `tests/test_basic.py`; test data in `tests/data/` (txt, csv, ROOT files).
- Optional deps: `boost-histogram`, `rich`, `uproot+awkward` — all exercised in tests.
- `ruff` is the sole linter/formatter (no black/isort). T20 (print) and NPY002 are allowed in tests.
- mypy targets `python_version = "3.10"`; pre-commit hook passes `--python-version=3.12` to handle numpy stubs.
